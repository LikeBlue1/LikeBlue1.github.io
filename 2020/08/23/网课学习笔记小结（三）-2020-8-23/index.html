二、Thymeleaf 的标准表达式及常见属性

1.标准表达式

标准变量表达式 选择变量表达式 URL表达式
语法 ${…} *{…} @{…}
用法 Thymeleaf 中的变量表达式使用 ${变量名} 的方式获取其中的数据例如：<span th:text="${user.name}">x</span> 选择变量表达式，也叫星号变量表达式，使用 th:object 属性来绑定对象<div th:object="${user}" > <span th:text="*{name}"张</span>等价于<span th:text="${user.name}" >张</span> URL表达式可用于 <script src="...">、<link href="...">、<a href="...">等，推荐使用相对URL方式：<a href="info.html" th:href="@{'/user/info?id='+${user.id}}">查看</a> （项目的上下文名会被自动添加）

2.常见属性

常见属性 作用 举例
th:action 定义后台控制器的路径，类似<form>标签的action属性 <form id="login" th:action="@{/login}">......</form>
th:each（重要） 从后台传来一个对象集合那么就可以使用此属性遍历输出，它与JSTL中的<c: forEach>类似，此属性既可以循环遍历集合，也可以循环遍历数组及Map <tr th:each="user: ${userlist}"><td th:text="${userStat.index}"></td><td th:text="${user.id}"></td><td th:text="${user.nick}"></td><td th:text="${user.phone}"></td></tr>
th:href 定义超链接 <a class="login" th:href="@{/login}">登录</a>
th:id 类似html标签中的id属性 <span th:id="${hello}">aaa</span>
th:if(重要) 条件判断 <span th:if="${sex} == 1" >男：<input type="radio" name="se" th:value="男" /></span>
th:switch/th:case（重要） switch，case判断语句 <div th:switch="${sex}"><p th:case="1">性别：男</p><p th:case="2">性别：女</p><p th:case="*">性别：未知</p></div>
th:object 用于数据对象绑定 通常用于选择变量表达式（星号表达式）
th:src 用于外部资源引入，比如<script>标签的src属性，<img>标签的src属性，常与@{}表达式结合使用 <script th:src="@{/static/js/jquery-2.4.min.js}"></script>
th:text（重要） 用于文本的显示 <input type="text" th:text="${realName}">
th:value（重要） 类似html标签中的value属性，能对某元素的value属性进行赋值 <input type="hidden" id="userId" th:value="${userId}">
